# 前端 UI 设计规范

> Vue 3 + Vite + Element Plus
> 风格参考：Obsidian（Wiki 浏览器）、ChatGPT（搜索问答）

---

## 一、整体布局

### 应用框架（所有页面共用）

```
┌─────────────────────────────────────────────────────────────┐
│  导航栏                                                       │
│  [🔖 LLM Wiki]  [KB切换下拉]  [搜索框]       [设置] [主题]   │
├──────────┬──────────────────────────────────────────────────┤
│  侧边栏   │  主内容区                                          │
│          │                                                   │
│  KB列表  │  （根据当前路由显示对应页面内容）                      │
│  Wiki树  │                                                   │
│  ・source│                                                   │
│  ・entity│                                                   │
│  ・concept                                                   │
│          │                                                   │
└──────────┴──────────────────────────────────────────────────┘
```

### 路由结构

```
/                          → 重定向到 /kb
/kb                        → KB 管理主页
/kb/{kb_id}/wiki           → Wiki 浏览器（首页：index.md）
/kb/{kb_id}/wiki/{type}/{name}  → 具体 Wiki 页面
/kb/{kb_id}/search         → 搜索问答
/kb/{kb_id}/graph          → 知识图谱
/kb/{kb_id}/pipeline       → 流水线管理
```

### 2.2 Wiki 浏览器（/kb/{id}/wiki）

**参考 Obsidian 设计，三栏布局**：

```
┌──────────────┬─────────────────────────────┬────────────────┐
│  文件树        │  Wiki 页面内容               │  本地关系图       │
│              │                             │                │
│  🔍 搜索文件   │  ← →  面包屑: entity/Claude  │  [关系图小图]    │
│              │  ─────────────────────────  │                │
│  📁 entity   │                             │  当前页：Claude  │
│   ├ Claude   │  # Claude                   │  ·── Anthropic  │
│   ├ OpenAI   │                             │  ·── agent      │
│   └ Hermes   │  > Anthropic 旗舰 AI 助手     │  ·── MCP        │
│              │                             │                │
│  📁 concept  │  ## 概述                     │  [在图谱中查看]  │
│   ├ agent    │  Claude 是 Anthropic 开发的   │                │
│   ├ mcp      │  大语言模型，以安全性和可用     │                │
│   └ rag      │  性著称...                   │                │
│              │                             │                │
│  📁 source   │  ## 核心特性                  │                │
│   ├ 文章1    │  - **多模态**：支持图像/文档    │                │
│   └ 文章2    │  - **长上下文**：200K tokens  │                │
│              │                             │                │
│              │  ## 关联关系                  │                │
│              │  - **所属公司**：[[Anthropic]] │                │
│              │  - **相关实体**：[[Claude Code]]│               │
│              │  - **核心概念**：[[agent]]    │                │
└──────────────┴─────────────────────────────┴────────────────┘
```

**交互细节**：
- `[[WikiLink]]` 点击 → 跳转到对应 entity/concept 页面
- 浏览历史：支持浏览器前进/后退
- 文件树：支持折叠/展开，点击直接跳转
- Markdown 渲染：代码高亮、表格、列表全支持
- 右侧小图：显示当前页面的一阶关联节点（5-8个）

┌────────────────────────────────────────────────────────────┐

│  ┌──────────────────────────────────────────┐ [搜索]       │

│  └──────────────────────────────────────────┘            │

│  使用 [[Hermes]] 实现持久化记忆主要通过两个文件：              │
│                                                            │

│                                                            │
│  ┌──────────────────┐  ┌──────────────────┐               │
│  │ 📄 concept/memory │  │ 📄 source/15-herm │               │
│  │ 记忆层，支持 FTS5  │  │ Hermes 进阶功能... │               │
│  │ 相关度：92%        │  │ 相关度：87%        │               │

- 回答中的 `[[WikiLink]]` 可点击，直接跳转 Wiki 页面
- 来源卡片点击「查看完整页面」→ 在 Wiki 浏览器中打开
- 支持历史记录（当前会话内）

## 三、主题与风格

### 颜色方案

```css
/* 主色 */
--color-primary: #409EFF;          /* Element Plus 蓝 */
--color-success: #67C23A;
--color-warning: #E6A23C;
--color-danger: #F56C6C;

/* Wiki 专用色 */
--wiki-entity: #409EFF;            /* 实体节点：蓝色 */
--wiki-concept: #67C23A;           /* 概念节点：绿色 */
--wiki-source: #9C27B0;            /* 来源节点：紫色 */
--wiki-link: #409EFF;              /* [[WikiLink]] 链接色 */

/* 暗黑主题 */
--bg-primary: #1a1a1a;
--bg-secondary: #252525;
--text-primary: #e8e8e8;
```

### 字体

- 正文：`-apple-system, 'PingFang SC', 'Microsoft YaHei', sans-serif`
- 代码：`'JetBrains Mono', 'Fira Code', monospace`

### WikiLink 渲染样式

```css
/* [[WikiLink]] 渲染为带样式的链接 */
.wiki-link {
  color: var(--wiki-link);
  background: rgba(64, 158, 255, 0.1);
  border-radius: 3px;
  padding: 0 3px;
  text-decoration: none;
  cursor: pointer;
}
.wiki-link:hover {
  background: rgba(64, 158, 255, 0.2);
  text-decoration: underline;
}
```

## 四、项目目录结构

```
web/
├── index.html
├── package.json
├── vite.config.ts
├── tsconfig.json
└── src/
    ├── main.ts                    # 入口
    ├── App.vue                    # 根组件 + 布局框架
    ├── router/
    │   └── index.ts               # 路由配置
    ├── stores/
    │   ├── kb.ts                  # Pinia：KB 列表/当前 KB
    │   └── wiki.ts                # Pinia：Wiki 浏览历史
    ├── api/
    │   └── index.ts               # Axios 封装 + 所有 API 调用
    ├── views/
    │   ├── HomeView.vue            # KB 管理主页
    │   ├── WikiView.vue            # Wiki 浏览器主视图
    │   ├── SearchView.vue          # 搜索问答
    │   ├── GraphView.vue           # 知识图谱
    │   └── PipelineView.vue        # 流水线管理
    └── components/
        ├── layout/
        │   ├── AppHeader.vue       # 顶部导航
        │   └── AppSidebar.vue      # 左侧文件树
        ├── wiki/
        │   ├── WikiTree.vue        # 文件树组件
        │   ├── WikiPage.vue        # Markdown 渲染
        │   ├── WikiMiniGraph.vue   # 右侧小关系图
        │   └── WikiLink.vue        # [[WikiLink]] 渲染
        ├── search/
        │   ├── SearchBox.vue       # 搜索框
        │   ├── SearchAnswer.vue    # AI 回答展示
        │   └── SourceCard.vue      # 来源引用卡片
        ├── graph/
        │   └── KnowledgeGraph.vue  # ECharts 图谱
        └── pipeline/
            ├── PipelineStages.vue  # 流水线阶段状态
            ├── ProcessLog.vue      # 实时日志
            └── DocList.vue         # 文档列表管理
```
